These are motor skills like walking, talking, and riding a bike that do not involve conscious thought and are usually difficult to explain verbally.

(A) Reflex

(B) Declarative Memory

(C) ** Procedural Memory

(D) Reaction

EXPLANATIONS BELOW

Concept note-1: -A procedural memory or implicit memory is often difficult to explain. If someone asked you how you drive a car or ride a bike, for example, you might struggle to put it into words.

Concept note-2: -Examples of procedural memories are knowledge of how to ride a bike or drive a car, how to play the drums, how to solve a puzzle, and how to walk.

Concept note-3: -Procedural memory is a part of the long-term memory that is responsible for knowing how to do things, also known as motor skills. As the name implies, procedural memory stores information on how to perform certain procedures, such as walking, talking and riding a bike.

Concept note-4: -When one acquires a new skill like riding a bicycle, the cerebellum is the part of the brain needed to learn the co-ordinated movement.

Concept note-5: -Skills such as playing an instrument or riding a bicycle are, however, anchored in a separate system, called procedural memory. As its name implies, this type of memory is responsible for performance.
