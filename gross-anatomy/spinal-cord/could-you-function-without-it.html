Concept note-1: -OR – =IF(OR(Something is True, Something else is True), Value if True, Value if False) NOT – =IF(NOT(Something is True), Value if True, Value if False)

Concept note-3: -Use the NOT function, one of the logical functions, when you want to make sure one value is not equal to another.
